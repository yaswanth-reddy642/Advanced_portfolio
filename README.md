# Advanced Portfolio Website

A modern and fully responsive animated portfolio website built using HTML, CSS, and JavaScript.  
Designed for showcasing projects, technical skills, achievements, and developer profiles professionally for internships and product-based companies.

---

# Features

- Modern UI Design
- Fully Responsive Layout
- Animated Hero Section
- Typing Text Animation
- Animated Background Effects
- Skill Progress Bars
- Project Showcase Section
- Smooth Scrolling Navigation
- Interactive Hover Effects
- Professional Dark Theme
- GitHub Ready Structure

---

# Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- CSS Animations

# Installation Steps

## 1. Clone Repository

```bash
git clone https://github.com/yaswanth-reddy642/Advanced_portfolio.git
```

---

## 2. Open Project Folder

```bash
cd Advanced_portfolio
```

---

## 3. Run Project

Open:

```bash
index.html
```

in browser.

OR

Use Live Server in VS Code.
